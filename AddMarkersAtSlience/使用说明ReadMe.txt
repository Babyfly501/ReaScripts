Core Function:
Add markers at slience for media items on selected tracks, mainly used for spliting vocal cilps

Instructions:
Open REAPER（ver 6.0 or above）
Select the tracks to be processed
Click “Actions - Show action list“ at the top of the screen
In the pop-up interface, click “New action - Load ReaScript”
In the file explorer that appears, locate “AddMarkersAtSilence.lua” and click Open

核心功能：
一键为选中的轨道内的音频片段的静音处添加标记，可用于语音分句

使用方法：
进入REAPER（ver 6.0以上）
选中需要处理的轨道
点击屏幕上方的Actions-Show action list
在跳出的界面点击New action-Load ReaScript
在跳出的资源管理器中找到AddMarkersAtSilence.lua并点击打开
