Reaper 批量区域重命名工具使用说明书

-导入方式

--点击桌面的Reaper图标进入Reaper

--在界面上方的工具栏找到并点击Actions→Show Action List

--在呼出的弹窗里点击New Action→Load ReaScript

--在资源管理器中找到BatchRegionRename.lua并确认

-系统要求

--REAPER 版本：6.82 或更高版本。

--依赖插件：需要安装 ReaImGui 扩展（可通过 ReaPack 安装）。

-界面与功能概览

工具界面自上而下分为以下几个核心模块：

--主题切换 (Theme)

提供 Dark (深色) 和 Light (浅色) 两种现代化主题。

您可以随时点击单选按钮进行切换，界面颜色会立即无缝自适应，以匹配您的工作习惯和视觉偏好。

--筛选与选择区 (Filter & Selection)

Filter (by name)：输入任意文本，下方的列表会实时筛选出包含该名称的区域。这对于在庞大项目中快速定位特定区域非常有用。

--Regions 列表：

显示项目中所有的区域（或筛选后的区域）。

列表支持实时预览：当您设置了修改规则后，列表中会以 原名称 -> 新名称 的格式直观显示修改结果。

核心选择逻辑：您可以勾选特定的区域来仅对它们进行修改。如果您没有勾选任何区域（全空状态），工具将默认对项目中的所有区域生效。

--控制按钮：

Select All：一键选中列表中的所有区域。

Clear Selection：一键取消所有选中状态。

Refresh：从项目中重新读取区域数据（如果您在 Reaper 中手动添加或删除了区域，可点击此按钮同步）。

-名称转换规则 (Transformations)

所有的名称修改将严格按照界面的自上而下顺序执行：

--Remove characters (移除字符)：

左侧框 (- / +)：从区域名称的开头删除指定数量的字符。

右侧框 (- / +)：从区域名称的结尾删除指定数量的字符。

提示：支持直接输入数字或点击加减号微调。清空输入框将默认视为 0。


--Find / Replace (查找与替换)：

Find (literal)：输入想要查找的字符。采用绝对精确匹配（支持符号如 ( ) % 等，无需担心代码转义错误）。

Replace with：输入替换后的字符。如果留空，则相当于删除查找到的字符。


--Prefix / Suffix (前缀与后缀)：

Prefix：在区域名称的最前方添加指定的文本。

Suffix：在区域名称的最后方添加指定的文本。

-执行操作

Apply Changes (应用修改)：点击后，上方设置的所有规则将立即应用到所选的区域上。

安全防误触：每次点击应用都会在执行后自动清空下方的所有规则输入框，防止下次操作时叠加遗留规则。

完美撤销：工具全面接入了 Reaper 的 Undo 系统。如果您对修改结果不满意，只需在 Reaper 中按下 Ctrl + Z (Mac 为 Cmd + Z) 即可瞬间撤销所有批量重命名操作。

-工作流示例

场景：清理渲染导出的尾缀并添加版本号 假设您有许多区域名称如下： Song_A_Render Song_B_Render

移除尾缀：在 Find 框中输入 _Render，Replace 框留空。（预览变为 Song_A）。

添加前缀：在 Prefix 框中输入 v2_。（预览变为 v2_Song_A）。

检查预览：确认上方列表中的预览 Song_A_Render -> v2_Song_A 无误。

应用：点击 Apply Changes 完成重命名。
